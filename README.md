Примерное описание базы данных:
Users - основная информация о пользователях
Exercises - таблица упражнений с их описанием и создавшим пользователем
Workout_daily_template - таблица, которая принимает несколько Exercises в формате json вместе с количеством подходов и повторений и создавшим пользователем
Workout_monthy_template - таблица, которая принимает около 30 пар day:Workout_daily_template в формате json, имеет информацию о выполняющем пользователе, моменте создания и текущем этапе. Каждый пользователь может иметь только один Workout_monthy_template (для первого раза).
